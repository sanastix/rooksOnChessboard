Завдання: 

Ще у дитинстві маленького Гаріка зацікавило питання: а скількома способами на шаховій дошці розміром n × n можна розставити n тур так, щоб вони не били одна одну.


Вхідні дані: 

Розмір шахової дошки - натуральне число, яке не перевищує 1000.

Вихідні дані:

Виведіть відповідь, знайдену Гаріком.
